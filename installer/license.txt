JAVA VERSION MANAGER
Gestore semplice e intuitivo per le versioni Java installate su Windows.

Questo software permette di:
• Visualizzare tutte le versioni Java installate
• Cambiare rapidamente la variabile JAVA_HOME
• Creare collegamenti sul desktop per un accesso rapido
• Gestire le versioni Java tramite interfaccia grafica

Requisiti:
• Windows 7 o superiore
• Almeno una versione di Java installata in C:\Program Files\Java\
• Privilegi di amministratore per modificare le variabili d'ambiente

Per supporto e aggiornamenti, visitare:
https://github.com/tuoutente/java-version-manager
